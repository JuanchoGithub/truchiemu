--- layout: default title: Building TruchiEmu description: How to build TruchiEmu from source --- # Building TruchiEmu This page is under construction. Building TruchiEmu from source requires: ## Prerequisites - macOS 14.0 or later - Xcode with Swift 5.9+ - XcodeGen (for project generation) ## Build Steps 1. Clone the repository 2. Run `xcodegen generate` to create the Xcode project 3. Open `TruchiEmu.xcodeproj` in Xcode 4. Build the project ## Dependencies - Libretro cores - SwiftData - Metal shaders See the main [repository README](https://github.com/JuanchoGithub/truchiemu) for complete instructions.
