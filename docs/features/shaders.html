Shader System

TruchiEmu's advanced shader system transforms your retro gaming experience with real-time visual enhancements, CRT simulation, and modern post-processing effects. Built on Apple's Metal framework, our shader pipeline delivers pixel-perfect rendering with zero performance overhead.

Service Overview

The shader system in TruchiEmu leverages custom Metal shaders to provide real-time visual enhancements for emulated games. Unlike traditional emulator filters, our shaders run directly on the GPU, ensuring smooth 60fps performance even on complex effects like CRT simulation and scaling algorithms.

Shader processing pipeline from emulation core through Metal to display

Core Features

  • Real-time Processing: All shaders execute on the GPU with sub-millisecond latency
  • CRT Simulation: Authentic phosphor glow, scanlines, and curvature effects
  • Scaling Algorithms: xBRZ, HQx, and custom integer scaling with aspect ratio preservation
  • Color Enhancement: LUT support, color correction, and HDR tone mapping
  • Performance Optimization: Automatic fallback to simpler shaders on lower-end hardware

Technical Architecture

The shader system consists of several key components:

  • Shader Compiler: Runtime compilation of .metal shader files with error handling
  • Effect Chain Manager: Stack multiple shaders with configurable ordering
  • Parameter System: Real-time adjustment of shader uniforms via UI controls
  • Fallback System: Automatic degradation to simpler shaders if compilation fails

Step-by-Step Configuration Guide

Basic Shader Setup

  1. Access Shader Settings

    Navigate to Settings → Video → Shaders in the TruchiEmu menu bar. The shader panel will display available presets and custom shaders.

  2. Select a Preset

    Choose from built-in presets like "CRT-Consumer", "LCD-Sharp", or "Original Pixel". Each preset includes pre-configured shader combinations optimized for different display types.

  3. Adjust Shader Parameters

    Use the parameter sliders to fine-tune effects in real-time. Changes apply immediately without requiring a restart.

  4. Test Performance

    Enable the FPS overlay (View → Show FPS Counter) to monitor shader performance. Complex shaders may impact frame rates on older hardware.

Advanced Shader Configuration

  1. Custom Shader Loading

    Place .metal shader files in ~/Documents/TruchiEmu/Shaders/. The shader manager will automatically detect and compile new shaders on launch.

  2. Shader Chain Creation

    Drag shaders in the effects chain to reorder processing. The output of each shader becomes the input for the next in sequence.

  3. Parameter Binding

    Bind shader uniforms to controller inputs for dynamic effects. For example, map scanline intensity to the right trigger for adjustable CRT effects during gameplay.

  4. Export Settings

    Save custom shader configurations as presets using File → Export → Shader Preset. Share presets with the community or across devices.

Configuration Examples

Example 1: Authentic CRT Television

// Shader Chain Configuration
type = "chain"
shaders = [
    "scale/xbrz-freescale",
    "crt/crt-geom", 
    "misc/scanlines"
]

// Parameter Settings
[params.crt-geom]
curvature = 0.15
corner_size = 0.02
sharpness = 0.8

[params.scanlines]
intensity = 0.3
thickness = 1.0

// Performance Settings
fallback_enabled = true
max_fps_impact = 5

Example 2: Handheld Console LCD

// Shader Chain Configuration
type = "chain"

    "lcd/lcd-grid",
    "color/color-correction"
]

// Parameter Settings
[params.lcd-grid]

[params.color-correction]

saturation_boost = 1.2

temperature = 6500

Example 3: Modern Upscaling

    "misc/anti-aliasing",
    "color/hdr-tonemap"
]

// Parameter Settings

model_path = "models/game_ai.pth"
denoise_level = 0.1

[params.hdr-tonemap]
peak_brightness = 400

Troubleshooting

Shader Compilation Failures

Symptom: Shaders fail to compile with "Metal compiler error" messages.

Solution: Check shader syntax compatibility. TruchiEmu requires Metal shader language 2.0+. Update macOS to the latest version for newest Metal features.

Performance Issues

Symptom: Frame rate drops when using complex shaders.

Solution: Enable performance mode in shader settings. Reduce shader chain complexity or lower internal resolution scaling.

Color Banding

Symptom: Visible color banding in gradients.

Solution: Enable dithering in the shader parameters. Increase bit depth to 10-bit if your display supports it.

Aspect Ratio Distortion

Symptom: Games appear stretched or squashed.

Solution: Configure aspect ratio settings in the shader parameters. Enable integer scaling for pixel-perfect display.

Shader Ordering Issues

Symptom: Effects appear in wrong order or produce unexpected results.

Solution: Reorder shaders in the effects chain. Remember: scaling should come first, then CRT effects, then color correction.

Custom Shader Loading

Symptom: Custom shaders not appearing in the menu.

Solution: Verify shader files are in the correct directory and have .metal extension. Restart TruchiEmu to refresh the shader cache.
