Controller Support

TruchiEmu provides comprehensive controller support with automatic mapping, per-game configurations, advanced deadzone calibration, and macro recording. Experience authentic gameplay with any controller from classic gamepads to modern arcade sticks, with full button remapping and turbo functionality.

Service Overview

The controller system in TruchiEmu offers native support for HID-compatible controllers with intelligent auto-mapping based on the connected device. Our advanced input processing includes individual deadzone calibration per-controller, analog stick curve customization, and macro recording for complex input sequences.

Input pipeline from controller hardware through mapping layer to emulation core

Core Features

  • Auto-Discovery: Instant recognition of USB and Bluetooth controllers
  • Smart Mapping: Automatic button mapping based on controller type and game system
  • Per-Controller Profiles: Individual settings for each connected controller
  • Advanced Calibration: Per-axis deadzone, sensitivity, and response curve adjustment
  • Macro Recording: Record and playback complex input sequences with timing
  • Turbo Functionality: Programmable rapid fire on any button with adjustable rate
  • Hotkey Support: Map controller buttons to emulator functions

Step-by-Step Configuration Guide

Basic Controller Setup

  1. Connect Controller

    Connect your controller via USB or pair via Bluetooth. TruchiEmu automatically detects the controller and shows a notification with the detected controller type.

  2. Verify Connection

    Open Settings → Controllers → Connected Devices to see your controller listed. Connected controllers show battery level if supported.

  3. Test Button Mapping

    Click "Test Controller" to open the input test panel. Press buttons and move sticks to verify all inputs register correctly.

  4. Adjust Global Settings

    In Settings → Controllers → General, configure global deadzone, vibration intensity, and LED settings for supported controllers.

Advanced Configuration

  1. Custom Button Mapping

    Go to Settings → Controllers → Button Mapping and select your controller. Click any button and press the desired input to remap. Save as profile for reuse.

  2. Analog Stick Calibration

    Access calibration in Settings → Controllers → Calibration. Adjust individual deadzones per axis and modify sensitivity curves to eliminate drift.

  3. Configure Hotkeys

    Map controller buttons to emulator functions in Settings → Controllers → Hotkeys. This allows save state creation, fast forward, and menu access from your controller.

  4. Enable Turbo Mode

    In Settings → Controllers → Turbo, enable turbo for buttons that need rapid fire. Set frequency for 5-30 presses per second based on your needs.

Troubleshooting

Controller Not Detected

Symptom: Controller connected but not showing.

Solution: Check macOS Privacy settings for input permissions. Try re-pairing Bluetooth controllers. Verify USB cable supports data.

Analog Stick Drift

Symptom: Character moves without input.

Solution: Increase deadzone to 15-20% in calibration settings. Clean controller sticks with compressed air. Run calibration wizard.

Mapping Not Saving

Symptom: Custom mappings reset on restart.

Solution: Save settings via main menu, not in-game. Create per-game profile in settings. Check config folder permissions.

Input Lag

Symptom: Delay between press and response.

Solution: Reduce audio buffer size. Use wired connection instead of Bluetooth. Disable many shader effects. Check CPU usage.

Vibration Not Working

Symptom: No force feedback.

Solution: Enable in controller settings. Check game supports vibration. Verify controller has adequate battery. Some cores require per-core enabling.

Hotkey Conflicts

Symptom: Controller triggers emulator functions accidentally.

Solution: Review hotkey settings. Use Game Focus Mode to disable hotkeys. Change combinations to avoid conflicts.
