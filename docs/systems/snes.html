Super Nintendo Entertainment System (SNES)

The Super Nintendo Entertainment System, released in 1990 (Japan) and 1991 (North America), represented a significant leap forward in console technology. At its heart was the Ricoh 5A22 CPU, a custom 16-bit processor based on the WDC 65C816, running at 3.58 MHz (2.68 MHz during memory access).

Hardware Overview

The system's graphics were powered by the Picture Processing Unit (PPU), actually composed of two chips: PPU1 and PPU2. This allowed the SNES to display 256x224 resolution (512x448 in high-resolution mode) with 32,768 colors available, though only 256 could be displayed simultaneously. The system supported up to 128 sprites on screen, with a maximum sprite size of 64x64 pixels.

Audio capabilities were revolutionary, featuring the Sony SPC700 audio CPU running at 1.024 MHz, paired with a 16-bit DSP. This configuration provided eight audio channels capable of producing high-quality stereo sound with advanced effects like echo, reverb, and pitch modulation.

Memory architecture included 128KB of RAM, 64KB of Video RAM, and 64KB of Audio RAM. The cartridge-based system allowed for additional enhancement chips, including the Super FX chip for 3D graphics, SA-1 for enhanced processing, and DSP chips for various mathematical operations.

ROM Format Support

Supported Formats

  • .sfc - Super Famicom (raw ROM dump, preferred)
  • .smc - Super Magicom (common, may have headers)
  • .fig - Multi Game Hunter format
  • .swc - Super Wild Card format
  • .bs - Satellaview (BS-X) downloads
  • .st - Sufami Turbo

Tip: Always prefer .sfc format when possible, as it has no headers or copier remnants. Many SMC files contain a 512-byte header you can strip with: dd if=input.smc of=output.sfc bs=512 skip=1

  • Runahead Frames: Enable 1-2 frames for reduced input latency
  • Audio Buffer: 64-128ms recommended
  • Apple Silicon: All cores run flawlessly, bsnes recommended for accuracy
  • Intel Integrated: Use snes9x with lighter shaders for 4K displays

Known Issues

Audio Crackling in DSP Games

Games using DSP enhancement chips (Pilotwings, Super Mario Kart) may have audio issues. Use snes9x core with "DSP Audio Fix" enabled.

Super FX Performance

Super FX games (Star Fox, Stunt Race FX) may run slowly. Enable "Super FX Overclock" in core options.

PAL Timing Issues

PAL region games may run at incorrect speed. Enable "PAL Mode" in video settings and set refresh rate to 50Hz.

MSU-1 Audio

Ensure PCM files are in the correct directory structure. Use snes9x or bsnes-mercury cores with MSU-1 support enabled.
