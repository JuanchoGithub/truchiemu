Nintendo Entertainment System (NES)

The Nintendo Entertainment System, released in 1983 (as Famicom in Japan) and 1985 (as NES in North America), featured a Ricoh 2A03 CPU running at 1.79 MHz (NTSC) or 1.66 MHz (PAL). This 8-bit processor was based on the MOS Technology 6502 architecture and included integrated sound generation capabilities.

Hardware Overview

The system's Picture Processing Unit (PPU) handled graphics rendering, supporting a resolution of 256x240 pixels with a color palette of 52 colors. The NES could display up to 25 colors simultaneously from its palette, with support for up to 64 sprites on screen at once.

Memory architecture consisted of 2KB of onboard RAM, with games utilizing additional memory mappers within cartridges to expand capabilities. The audio system featured five channels: two pulse wave channels, one triangle wave channel, one noise channel, and a DPCM sample channel for digital audio playback.

ROM Format Support

Supported Formats

  • .nes - iNES format (most common)
  • .nez - Compressed NES format
  • .unf/.unif - Universal NES Interchange Format
  • .fds - Famicom Disk System images

Mapper Support

Mapper compatibility is crucial for NES emulation. TruchiEmu supports over 100 mappers, including:

  • Mapper 0 (NROM) - Early titles like Super Mario Bros.
  • Mapper 1 (MMC1) - Games like The Legend of Zelda, Metroid
  • Mapper 4 (MMC3) - Super Mario Bros. 3, Mega Man series
  • Mapper 7 (AxROM) - Rare early titles

Best Shader Recommendations

CRT / Retro Authenticity

  • CRT-Royale: Advanced CRT simulation with phosphor glow and scanlines
  • CRT-Geom: Geometric CRT simulation with curvature and scanlines
  • NTSC-256: Replicates composite video artifacts and color bleed

Modern Enhancements

  • ScaleFX: Smooth scaling with edge preservation
  • xBRZ: High-quality upscaling for pixel art
  • HQx: Classic pixel art smoothing algorithm

Performance Optimization

NES emulation is generally lightweight. Recommended settings:

  • Frameskip: 0 (auto) for most systems
  • Audio Buffer: 64ms for low latency
  • Video Sync: Enabled for tear-free display

Platform Notes: All cores run at full speed on Apple Silicon. Intel integrated graphics users should use lighter shaders like ScaleFX or HQ2x.

Known Issues

Audio Popping

Occasional audio distortion during scene transitions. Increase audio buffer to 96-128ms to resolve.

Sprite Flickering

This is accurate NES behavior (8 sprite per scanline limit). Enable "Remove Sprite Limit" in core options if desired.

Save State Compatibility

Save states created with one core may not work with another. Use the same core for loading that was used for saving.

Famicom Disk System

Ensure you have the correct BIOS file (disksys.rom) placed in the system directory.
