MAME - Multiple Arcade Machine Emulator

MAME (Multiple Arcade Machine Emulator) is a preservation project covering thousands of arcade systems. Instead of emulating a specific hardware platform, MAME emulates the original arcade hardware, ROMs, and CPU architectures of countless arcade machines spanning from 1975 to the modern era.

ROM Format Support

ROM Set Types

  • Split: Common files shared across clones
  • Non-Merged: All files in single archive
  • Merged: Parent and clones combined

Compression Formats

  • .zip - Standard compression
  • .7z - Advanced compressed archives
  • .chd - Compressed hard disk for media (CDs, HDD)

BIOS Requirements

Many systems require specific BIOS files in the system directory: neogeo.zip for Neo Geo, cps2.zip for Capcom CPS2 games, and various others for different hardware sets.

Best Shader Recommendations

Choose shaders based on the original display technology of your arcade game:

  • Early Raster Games (1975-1990): CRT-Geom, CRT-PI for scanlines
  • Late 80s/Early 90s: CRT-Royale-Aperture, MDAPT for dithering
  • 3D Games (Mid-90s+): FXAA/SMAA for edge smoothing, ScaleFX for upscaling

Performance Optimization

MAME's accuracy comes at a significant CPU cost:

  • z80-based games: Low CPU requirement
  • 68k-based games: Moderate CPU
  • 3D hardware: High CPU, GPU accelerated rendering preferred

Speed Optimization

  • Frame skipping: 0 or 1 maximum
  • Audio latency: 128-256ms
  • DRC (Dynamic Recompiler): Enable when available
  • Use HLE over LLE for performance when accuracy is less critical

Known Issues

ROM Set Mismatch

Games fail to load with "missing files" errors. Ensure ROM version matches MAME core version exactly. Use ROM managers like ClrMamePro to verify sets.

CHD Format Nuances

Use chdman tool from MAME distribution to create CHDs. Store in path matching MAME's expected structure.

Vector Games Performance

Vector games (Asteroids, Battlezone) may run slowly at high resolutions. Lower internal resolution and enable antialiasing for vector beam simulation.

Save State Support

Some drivers lack save state support. Use nvram saves for games with modified NVRAM storage.
