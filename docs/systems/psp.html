PlayStation Portable (PSP)

The PSP, released in 2004, packed console-quality hardware into a handheld form factor. Central processing came from a MIPS R4000-based CPU running at 333 MHz, with a dedicated GPU handling 3D graphics acceleration capable of rendering approximately 33 million polygons per second.

Hardware Overview

The GPU supported transform and lighting with hardware clipping, and featured a 512-bit floating-point engine with vertex and pixel processing capabilities. The display was a 4.3-inch TFT LCD with 480x272 resolution and 16.77 million colors.

Memory architecture included 32MB main RAM (64MB in PSP-2000 and later models), 4MB embedded DRAM, and Memory Stick PRO Duo expansion slots. The system featured both Universal Media Disc (UMD) drive for physical games and digital distribution capabilities.

Best Shader Recommendations

  • FXAA: Anti-aliasing for 3D games
  • SMAA: Superior edge detection
  • Sharpen: Enhance texture clarity at higher resolutions

Rendering Resolution

  • 2x PSP (960x544): Default enhancement
  • 3x PSP (1440x816): Good balance of quality and performance
  • 4x PSP (1920x1088): High quality, requires decent GPU
  • 5x+ PSP: Extreme quality, powerful GPU needed

Core Speed Hacks

  • I/O Timing: Set to Fast for most games
  • CPU Clock: Auto (333MHz) typically sufficient

Hardware Requirements

  • Apple Silicon: M1/M2 handle up to 10x resolution flawlessly
  • Intel Integrated: 2-3x resolution max
  • GPU Memory: 2GB+ VRAM for 4x+ rendering

Known Issues

Networking Support

Ad-hoc multiplayer features generally unsupported. Some local network functionality works via PPSSPP's experimental adhoc.

UMD Video

UMD Video discs are not supported. Extract video content separately using PC utilities.

Save Encryption

PPSSPP handles encryption transparently. Use DecryptSave tool if extracting saves for real PSP use.

Homebrew Compatibility

Enable "PSP-2000/3000" model setting for expanded homebrew compatibility.
